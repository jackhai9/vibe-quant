# src 目录说明

核心业务代码入口与模块实现。<br>
按功能拆分为多个子包。<br>
模块间以 `models.py` 数据结构协作（含成交角色/盈亏/手续费）。

## 文件清单

- `main.py`：应用入口与生命周期管理
- `models.py`：核心数据结构与枚举
- `__init__.py`：根模块导出
- `config/`：配置加载与模型
- `exchange/`：交易所适配器
- `ws/`：WebSocket 客户端
- `signal/`：信号引擎
- `execution/`：执行引擎
- `risk/`：风控与保护止损
- `notify/`：通知模块
- `utils/`：工具与日志
