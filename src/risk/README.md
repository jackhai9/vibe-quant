# src/risk 目录说明

风控与限速模块。<br>
计算强平距离并触发兜底。<br>
维护保护止损（algo 撤单）、外部止损有效性检查与限速器。

## 文件清单

- `manager.py`：风控判断与限速触发
- `protective_stop.py`：保护性止损维护
- `rate_limiter.py`：滑动窗口限速器
- `__init__.py`：模块导出
